We are excited to demonstrate the new Comunica MCP tooling at the ESWC 2026 conference in Croatia.
[This work](https://rubensworks.github.io/article-eswc2026-demo-comunica-mcp/) showcases how Comunica can be exposed as a Model Context Protocol (MCP) server, enabling AI agents to directly query Linked Data and SPARQL endpoints through a standardized interface.
By leveraging MCP (an open standard that connects AI systems to external tools and data sources) this approach allows agents to access and reason over decentralized RDF knowledge on the Web in a more flexible and interoperable way.
Our demonstration highlights how this integration brings together semantic web technologies and modern AI tooling to enable richer, real-time data-driven applications.
